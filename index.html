Projects

Look at all this neat stuff I made!

A freecodecamp assignment, this app generates a random quote at the push of a button. It also lets you tweet it.

This is a clock inspired by the Javascript30 challenge. Only vanilla javascript was used - no jQuery! CSS animations move the hands according to the time. I may go back and add a stopwatch.

This is a template for a political or marketing campaign. There is a form for subscribing to the mailing list. Several media queries were used to make sure the site looked optimal on the greatest number of devices. Using rem values for the fonts made the media queries much more concise.

A mobile-responsive web site based on a tutorial at w3newbie.com. It was one of my first forays into learning responsive design. The content is populated with my poor excuse for a sense of humor.

My first Freecodecamp project, this is a tribute page for the great muslim explorer, Ibn Battuta. If you are interested in learning more, I highly recommend the book The Adventures of Ibn Battuta, by Ross E. Dunn.

As for the site itself, it's pretty much just a landing page. Since it's bare-bones HTML and CSS, I leaned heavy on visual design, as opposed to functionality. I also used a beautiful free font called Reem Kufi for all the text - including the arabic.
